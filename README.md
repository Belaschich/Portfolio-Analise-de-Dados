# 📊 Retail Sales Analysis – Projeto de Portfólio

## 🎯 Objetivo
Analisar performance de vendas em shoppings centers.

## 🛠 Ferramentas Utilizadas
- Python (Pandas)
- SQL (SQLite)
- Power BI

## 📌 Principais Insights
- Shopping com maior faturamento: Mall Of Istanbul
- Categoria mais lucrativa: Roupas
- Forma de pagamento predominante: Dinheiro
- Ticket médio: R$ 841,97
